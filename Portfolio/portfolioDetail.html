Portfolio

Beschrijving project

Dit project was een taak van het vak "User Interfaces". De bedoeling van dit project is het maken van een site met gebruik van semantische HTML en gebruik van correcte CSS. De website is verdeeld in 5 Onderdelen. De "Home" pagina met een kleine introductie. Een "over mij" pagina met een beschrijving van mezelf en mijn interessen en ambities. "projecten" een pagina met twee projecten dat ik zelf heb gemaakt en een detailpagina van beide projecten. Een blanco pagina voor het vak "werkplekleren" dat later verder ontwikkeld zal worden. Ten laatste nog een "contact" pagina waar bezoekers van de site contact met mij kunnen opnemen.

Evolutie

Dit project heeft meerdere fasen van ontwikkeling gekend. Ten eerste ben ik begonnen met het maken van een moodboard met verschillende foto's en designs die mij zouden inspireren in het bedenken van een design. Nadat ik een design kon inbeelden, ben ik begonnen met het tekenen van wireframes met gebruik van het programma "figma". Wanneer de wireframes op punt waren ben ik dan uiteindelijk begonnen met het ontwikkelen van de website. De website heb ik in fasen gemaakt. Ik ben eerst begonnen met de homepagina en daarna ben ik daaruit verder gegaan tot de website compleet was. Hieronder heb je een link naar de github repository waar je de code van de website kan bekijken en ook screenshots waar je het resultaat kan zien a.d.h.v. screenshots.
