MovieSearcher

Beschrijving project

Dit project was een soort van bijles voor mezelf. Ik wou namelijk leren werken met de framework "react". Ik heb de basis geleerd van react en heb daarna een applicatie gemaakt om films op te zoeken. de site zelf is zeer gelimiteerd. Het enigste waar het voor dient is om te kunnen nakijken of een bepaalde film bestaat en als het bestaat kan je het jaar dat het uitkwam zien.

Evolutie

Al is de site niet zo interessant, de evolutie ervan is dat in ieder geval wel. aangezien ik nog nooit had gewerkt met react was de eerste fase van het ontwikkelen van deze site studeren. ik heb urenlang videos op youtube bekeken, cursussen gevolgd op het internet. Wanneer ik het gevoel had dat ik genoeg kennis had om iets te ontwikkelen ben ik begonnen met de design van de website te bedenken. Daarna ben ik begonnen om die design over te zetten naar code. Nadat de design klaar was moest ik een API oproepen om de data van de films te kunnen krijgen. Ik heb hiervoor de API gebruikt van "omdb". nadat de API klaar was, moest ik de onderliggende logica schrijven om de door de user gekozen film(en) op te zoeken en weer te geven op het scherm. Hieronder kan je de link klikken om de code zelf is te bekijken en ook screenshots bekijken van de website.
